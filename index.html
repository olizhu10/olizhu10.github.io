Research Intern @ Cornell Tech Language, Interaction, and Learning Lab

May 2019 – August 2019

Here, I found a great foundation for my development and NLP knowledge as I was exposed to many different technologies and was given ample amount of freedom in my work. In my first project, I contributed to Cereal Bar, a game environment used to collect natural language data about collaborative interactions. It was built using Unity, C#, JavaScript, and Python. I enhanced the system by creating a feedback mechanism, so that the guiding user could provide real-time feedback to the following user they are collaborating with. To further extend the use of the game environment, we wanted to develop an add-on to be used for second language acquisition. I expanded the game to include a translation option using the Google Translate API. I also prepared it for a study to see how well it could be used as a second language acquisition tool. This included updating the deployment mechanism to allow participants to be manually paired, rather than randomly, and updating the server, database, and reporting html files to gather and record feedback.

In my second project, I worked with a partner to begin a potential study from scratch. We analyzed the Newsroom dataset of 1.3 million news articles and human-written summaries to create a subset of news articles paired with multiple reference summaries each with different properties. I investigated data first-hand by manually annotating a small subset of data to see if it was viable to cluster data based on event, and then find a diverse set of summaries for a single event. After proving viability, I clustered data using DBSCAN with TF-IDF score as the clustering metric to identify news articles reporting on the same event. Simply clustering data by event was rather difficult, as similar events like soccer matches at different tournaments were clustered together, unwantedly, as well as other nuanced problems. We tested various different methods and alterations to our algorithm to address these. I also created a simple web application using HTML/CSS, JavaScript, Flask and Python to display and easily interact with the clustered data. This project would address an unsolved problem in (then current) NLP. There are models out there that are able to summarize articles, but none that can successfully produce summaries with specific writing styles. By curating this dataset of articles paired with multiple diverse human-written summaries, models could be trained to summarize with particular levels of extractiveness or abstractiveness.

During my experience here, I was able to successfully navigate my first projects outside of a class environment. The learning curve was steep– navigating a codebase and Git for the first time, learning NLP concepts, using Python libraries– but it was one of my most rewarding experiences. I felt empowered to work on complex projects and supported to dive into unexplored territory. As a computer science student with only two classes of computer science under my belt and no experience coding outside of class, dropped into an unfamiliar environment, I gained resilience and confidence.
